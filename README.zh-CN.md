# saveiOS6

给老设备多一点时间。

这是一个为 `iOS 6` 和更老越狱设备准备的小型 Cydia 源项目。  
它提供一个尽量轻量的网页入口、证书下载资源，以及基础的软件源目录结构。

## 项目内容

- 简洁的首页
- 根证书与 `mobileconfig` 资源
- `Packages` / `Packages.gz`
- `debs/` 软件包目录

## 这个项目想做什么

很多旧设备不是坏了，只是跟不上今天的互联网了。

证书老了，TLS 变了，网页越来越重，很多原本还能用的设备就这样慢慢失去连接能力。  
这个项目想做的事情很简单：尽可能用朴素、稳定、可维护的方式，让这些设备继续活着。

## 原则

- 保持简单
- 优先兼容老设备
- 少脚本，少依赖
- 不做过度设计

## 说明

- 这个项目不保证所有现代网站都能恢复访问
- 即使安装了新证书，部分网站仍可能因 TLS 配置过新而无法连接
- 证书资源和源内容需要长期维护

如果你也还在给老设备续命，这个仓库就是写给你这样的人的。
